Sən BioScript adlı biometrik səhiyyə idarəetmə sisteminin Həkim Paneli üçün PyQt5 GUI proqramı hazırlayan süni intellektsən. Proqram barmaq izi ilə pasiyent tanıma, resept yazma, pasiyent tarixçəsinə baxış, həkim analitikası, və MySQL əsaslı idarəetmə funksiyalarını özündə birləşdirir.

Proqramda AS608 barmaq izi modulu istifadə olunur və biometrik tanıma prosesi server tərəfində aparılır. Aşağıdakı bütün texniki və funksional tələblərə əməl et:

🩺 Giriş Paneli:
doctors cədvəlindən istifadə edilir

Giriş istifadəçi adı və parol ilə edilir (username, password)

Uğurlu girişdən sonra həkimə yalnız öz dashboard və öz yazdığı reseptlər göstərilir

🧬 Barmaq İzi ilə Tanıma (Serverdə):
Barmaq izinin tanınması cihazda yox, server tərəfində baş verir.

AS608 cihazı barmaq izini oxuyur və DownChar() + UploadTemplate() ilə template çıxarır

Template base64 formatında çevrilərək MySQL bazasındakı patients.fingerprint_template (BLOB) sahəsinə yazılır

Python GUI bu template-i serverə göndərir

Serverdə NBIS (NIST Biometric Image Software) və ya VeriFinger kimi biometrik SDK-lar vasitəsilə tanıma (matching) prosesi aparılır

Əgər uyğun pasiyent tapılarsa:

Onun patient_id GUI-yə qaytarılır

GUI həmin pasiyentin məlumatlarını göstərir

Əgər uyğunluq tapılmazsa:

Yeni pasiyent qeydiyyat paneli açılır

Qeydiyyatdan sonra həmin template fingerprint_template sahəsinə yazılır

📋 Yeni Resept:
Barmaq izi tanınandan sonra prescriptions cədvəlinə yeni resept yazılır:

doctor_id, patient_id, hospital_id, complaint, diagnosis

prescription_items cədvəli ilə dərmanlar daxil edilir:

name, dosage, instructions

🕓 Pasiyent Tarixi:
Barmaq izi ilə pasiyent tanınır → GUI həmin pasiyentin bütün reseptlərini göstərir

Göstərilən sahələr:

Dərman adı, dozası, istifadə qaydası, şikayət və diaqnoz

📊 Həkim Dashboardu:
Yazdığı resept sayı (günlük, aylıq)

Aktiv resept sayı

Statistik vizual qrafiklər (matplotlib, PyQtChart və s.)

🔧 Texniki Tələblər:
GUI: PyQt5

Dil: Azərbaycan dili

Biometrik tanıma: NBIS və ya VeriFinger SDK ilə tanıma

Tanıma prosesi serverdə aparılmalıdır (Python və ya C++ vasitəsilə NBIS inteqrasiyası ilə)

Template uyğunluğu tapıldıqda patient_id GUI-yə qaytarılır

Bu sistemi maksimum təhlükəsizlik, sürət və real-biometrik uyğunluqla kodlaşdır. Fokusun GUI-nin axıcı işləməsi, MySQL ilə düzgün sinxronizasiya və pasiyent məlumatlarının düzgün idarə edilməsidir.
